💪 헬스 트래커는 운동 루틴 관리, 기록, 통계를 한 곳에서 관리할 수 있는 앱이에요.
데이터는 브라우저 localStorage에 저장됩니다.

🏋️ 라이브러리 사용법

• 상체 / 하체 / 유산소 탭으로 구분돼요.
• ✏️ 버튼으로 운동 이름, 부위, 팁, 코멘트를 수정할 수 있어요.
• 🗑 버튼으로 삭제하면 루틴에서도 자동으로 제거돼요.
• 코멘트는 개인 메모나 주의사항을 적어두는 용도로 활용하세요.

📋 루틴 사용법

1. 라이브러리에서 운동을 먼저 등록하세요.
2. 루틴에서 + 루틴 추가 → 운동 추가로 구성하세요.
3. 루틴의 각 운동 옆 기록 버튼을 누르면 바로 기록 화면으로 이동해요.
4. ✏️ 버튼으로 루틴 이름 수정, 🗑 버튼으로 루틴 삭제가 가능해요.

✏️ 기록 사용법

1. 날짜를 선택하고 운동 선택하기 버튼으로 오늘 할 운동을 골라요.
2. 세트별로 무게와 횟수를 입력하세요.
3. + 세트 추가를 누르면 이전 세트 값이 자동 복사돼요.
4. 유산소 운동은 시간/거리/경사를 입력하세요.
5. 저장 버튼을 꼭 눌러야 기록이 저장돼요!

📦 백업 & 복원

✅ 수시로 엑셀 백업을 받아두세요!
히스토리 탭 → 📊 내보내기 버튼으로 전체 데이터를 백업할 수 있어요.

백업 파일에는 아래 3가지가 모두 포함돼요:
• 운동기록 — 날짜별 세트/무게/횟수
• 라이브러리 — 등록된 운동 목록
• 루틴 — 루틴 구성 정보

복원할 때는 히스토리 탭 → 📂 업로드 버튼을 누르면 돼요.

⚠️ 꼭 읽어주세요 (데이터 관리)

🔴 브라우저 데이터를 삭제하면 모든 기록이 사라져요.
🔴 다른 기기나 브라우저에서는 데이터가 공유되지 않아요.
🟡 크롬 → 사파리 등 브라우저를 바꾸면 데이터가 안 보여요.
🟡 시크릿 모드에서는 데이터가 저장되지 않아요.
